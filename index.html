IK Solver Technical Details

Method: Damped Least Squares (DLS)
Max Iterations: 500 per solve call
Damping Factor (λ): 0.025 (regularization)
Convergence Criterion: ||error|| < 1e-5 meters
Step Size (α): 0.35 (gradient descent)
Joint Limits: Enforced via clamping per joint
Error Vector: [Δx, Δy, Δz, 0, 0, 0] (position only)
Solver: Gaussian elimination with partial pivoting

Analysis & Calibration Tabs

Joints Tab: Real-time 6xN Jacobian matrix display with joint space bar visualization showing position within limits
Dynamics Tab: Mass matrix M(q) approximation, gravity vector G(q), torque equation τ = Mq̈ + C(q,q̇)q̇ + G(q)
Calibration Tab: DH error model with δθ, δd, δa, δα tolerance parameters and RMS position error estimation
